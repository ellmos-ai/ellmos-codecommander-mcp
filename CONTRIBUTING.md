# Contributing to ellmos-codecommander-mcp

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone: `git clone https://github.com/YOUR_USERNAME/ellmos-codecommander-mcp.git`
2. Install: `npm install`
3. Build: `npm run build`

## How to Contribute

- **Bug fixes**: Open an Issue or submit a PR directly
- **New code analysis tools**: Extend the tool set
- **Language support**: Better parsing for more languages
- **Documentation**: Always appreciated

### Good First Issues

Look for [`good first issue`](https://github.com/ellmos-ai/ellmos-codecommander-mcp/labels/good%20first%20issue) labels.

## Guidelines

- TypeScript with strict types
- One feature/fix per PR
- Test with real codebases before submitting
